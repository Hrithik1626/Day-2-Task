Difference Between documents and window objects?

Documents:
         1:The document represents the HTML document that is displayed in the window. 
         2:The document object has many properties and methods that allow us to manipulate HTML document
         3:use the element's properties and methods to change its content or appearance.
         4:Properties-getElementById(), getElementsByTagName().
         5:Methods-setAttribute(), removeAttribute(), getAttribute()




window objects:
        1:The window object represents the browser window.
        2:The window object has many properties and methods that allow us to interact with the browser window
        3:Properties-location, href, history.
        4:Methods-open(), close(), resize().
